Style Library (local only)
==========================

Put sermon series reference graphics here as `.jpg`, `.png`, or `.webp` files.

- You can add as many files as you want.
- Optional subfolders are supported for organization (for example: `minimal/`, `photo/`, `illustration/`, `abstract/`).
- These files are local working references and are not committed to git.
- The app auto-scans this folder recursively when building Style Library Mode options.

Rights notice:
- Only use style reference images you have rights or license to use.
- The generator should create original compositions, not copies.
